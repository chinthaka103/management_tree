# management-tree
Algorithm to create a management tree in java
# How to Build and Run

#### Clone Repository
<pre>
  git clone https://https://github.com/chinthaka103/management_tree.git
</pre>

#### Build Application
<pre>
  cd KiwiPlanAssessment <br />
  mvn clean install
</pre>

#### Run Application
<pre>
  cd target <br />
  java -jar kiwiplan-0.0.1-SNAPSHOT.jar
</pre>
